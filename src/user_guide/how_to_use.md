# 如何使用

本文将介绍在安装完成**MetaMystia Mod**后，您在游戏中应如何进行基本操作与联机。

如果您不熟悉电脑网络或“命令”这些概念，也不用担心，只要照着步骤操作，就可以和朋友成功连线游玩。

## 游戏内快捷键

在正常游玩过程中，MetaMystia提供了一些额外的按键，用来查看联机状态和操作提示。

- **按<kbd>\\</kbd>键**：显示或隐藏左下角的联机状态提示。

这个提示主要用来告诉您：

> Mod是否正常运行、当前有没有开启联机、有没有成功连接到朋友。

如果这里什么都不显示，通常说明Mod没有正常加载。

## 游戏内控制台

MetaMystia内置了一个**游戏内控制台**，用于输入联机相关命令。

### 打开与关闭方式

- **按<kbd>/</kbd>键**：打开控制台
- **按<kbd>Enter</kbd>键**：确认并发送您输入的内容
- **按<kbd>ESC</kbd>键**：关闭控制台

控制台仅在游戏运行时可用，不会影响正常的游戏操作。

## 常用联机命令

在控制台中，您可以使用以下常见的联机指令：

- `/mp start server`启动联机服务（作为房主使用）
- `/mp stop`停止当前联机服务
- `/mp connect <IP地址>`连接到指定IP地址的联机房间（请将`<IP地址>`替换为房主的实际IP，不要用尖括号包裹）
- `/mp disconnect`断开当前连接
- `/mp status`查看当前联机状态
- `/mp kick uid 114`踢出房间中uid=114的玩家
- `/mp kick id MetaMiku`踢出房间中名为MetaMiku的玩家
- `/mp id MetaMiku`将自己的ID设置为MetaMiku
- `/mp maxplayers 4`将联机房间的人数上限改为4（默认为2）

> [!TIP]
> 为保证安全性，启用联机服务后，默认只允许有且只有1位玩家可以加入，如果您需要更多玩家加入，请使用`/mp maxplayers`命令调整人数上限。

## 切换皮肤

所有联机玩家都使用米斯蒂娅可能导致难以区分，为此我们提供了**皮肤系统**，允许玩家切换为不同的服装以便区分，具体请参见[皮肤系统](./skin.md)。

## 联机方式与网络要求说明

MetaMystia当前使用的是**P2P（点对点）联机方式**，即玩家之间**直接建立连接**，不经过官方服务器，不支持本地同屏。因此，在联机前需要注意以下网络条件：

- **双方在同一局域网（如同一Wi-Fi）**

    可直接使用房主的局域网IP进行连接。

- **不在同一局域网时**

    可以使用**Tailscale**、**ZeroTier**等虚拟局域网工具，让双方处于“看起来像同一局域网”的环境中，再使用对应的虚拟IP进行连接。

- **双方都有公网IP**

    可以直接使用房主的公网IP进行连接（对网络环境要求较高，不推荐新手优先尝试）。

> [!TIP]
> MetaMystia内置的TCP服务器默认监听`40815`端口，如果您正在使用内网穿透服务，您可能需要映射该端口。

### 连接超时或无法连接时的排查建议

如果出现**连接超时或无法连接**的情况，请优先检查：

- 主机是否成功执行`/mp start`，而不是`/mp start server` 等错误命令，高级玩家可以使用 `/mp start 25565` 来指定 TCP 监听端口
- 客机是否在主机开启联机后成功执行`/mp connect <IP地址>`（不含尖括号），而不是`/mp connect`、`/mp start client`或`/mp start cilent`等错误命令
- IP是否正确，主机与客机之间是否可以正常通信，**防火墙**是否阻止了游戏的网络通信
  （可尝试临时关闭防火墙，或为游戏添加放行规则）

为了更快定位问题，**提问时请务必包含以下信息**：

    如：Steam最新正版、4.3.0c学习版

    如果不是Windows 10 x64或Windows 11 x64，请注明

- 网络服务说明

    如果使用了内网穿透、虚拟组网等服务，请附带相关日志
